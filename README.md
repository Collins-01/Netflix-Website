# Netflix Website Clone with React.JS





## Get Started

#### `Git Clone` [http://localhost:3000](http://localhost:3000)

#### `npm install ` To install all dependencies

#### `npm start` To run the project.






### Features
 - Authentiction
 - Fetch list of movies from TMDB Api.

### Tools

- ReactJS
- Typescript
- Firebase v9
- Redux
- Axios
